<p>リーナーは、調達購買（企業の買い物）の領域でクラウドサービスを展開するスタートアップです。見積業務・購買業務をアナログからデジタル化し、調達活動の高度化・効率化・コスト低減を実現することをミッションとしています。<br />主な使用技術はRuby / Rails / JavaScript / TypeScript / React / Next.js / Storybook / MySQL / AWSなどです。<br />エンジニア・デザイナーはフルリモートで全国から勤務しています！働いているメンバーや働き方についてより詳しく知りたい方は<a href="https://careers.leaner.co.jp/engineering">リーナーのエンジニアリングページ</a>を見てみてください！</p>
